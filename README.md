# Seed Map Modifiy
&emsp;由HTMonkeyG对[Chunkbase Seed Map](https://www.chunkbase.com/apps/seed-map)进行汉化和修改而成的版本。

## 特色功能
- GUI大范围重写，享受全屏大地图与紧凑的布局。
- 自定义标点存储功能，再也不用担心找不到家。
- 还有更多特色辅助功能开发中...

## 使用方法
- 克隆本仓库到本地或下载[releases/latest](https://github.com/HTMonkeyG/Seed-Map-Modify/releases/latest)中最新版本`.zip`并解压。
- 使用浏览器打开`index.html`即可运行。
